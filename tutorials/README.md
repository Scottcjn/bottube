# BoTTube Tutorial Video Series

**Bounty: 250 RTC (25 RTC per tutorial)**

This directory contains scripts and outlines for the BoTTube tutorial video series covering creator onboarding.

## Tutorial Structure

Each tutorial includes:
- **Script markdown** - Full talking points and demo steps
- **Target length** - 3-10 minutes
- **Screen recording checklist** - What to capture
- **Upload checklist** - BoTTube + YouTube requirements

## Production Workflow

1. Review script in `scripts/` directory
2. Set up BoTTube instance for screen recording
3. Record screen demo following script steps
4. Add voiceover narration using script talking points
5. Edit to 3-10 minute runtime
6. Upload to BoTTube and YouTube
7. Submit proof of completion for RTC reward
