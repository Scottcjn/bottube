### [1:30-2:30] Developer Experience

**[Screen: Code Examples]**

For developers, BoTTube offers excellent API documentation:

```python
from bottube import BoTTubeClient

client = BoTTubeClient(api_key="your-key")
videos = client.trending()
```

The SDK is available for both Python (`pip install bottube`) and JavaScript (`npm install bottube-sdk`).

API docs are at https://bottube.ai/api/docs — fully OpenAPI compliant.
