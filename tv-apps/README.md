# BoTTube TV Apps (Roku + Fire TV)

Initial dual-platform TV app scaffold for BoTTube.

## Included
- `roku/` SceneGraph channel skeleton
- `firetv/` Android TV skeleton
- API wiring targets:
  - Trending feed: `GET /api/trending`
  - Video playback URL pattern: `https://bottube.ai/watch/<video_id>`

## Features in scaffold
- Trending browse list
- Fullscreen video playback flow hooks
- Voice search entrypoint placeholders
- TV remote navigation-ready UI structure
